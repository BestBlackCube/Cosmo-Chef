# Cosmos-Chef 클라이언트 개발 1.0.0v
## 기능 설명
게임의 스토리를 보기 위해서는 플레이어는 퍼즐을 풀어야 하는 요소가 존재한다.  

플레이어가 특정 구간을 지나가게 되면 TMPro를 사용해서 스토리를 텍스트로 출력하며, 그저 자동으로 게임이 진행된다면 플레이어가 게임을 왜 해야하는 의문점이 들기 때문에 다음 스테이지를 넘어가기 위해서는 퍼즐을 풀어 다음 스토리를 보러가는 기믹을 추가한다.

## 버전 표기법 (Semantic Versioning)
```
[주 버전].[부 버전].[수 버전]
   0   .   0   .   0

- 주 버전: 하위 호환성이 깨지는 변경
- 부 버전: 하위 호환성 유지하며 기능 추가
- 수 버전: 하위 호환성 유지하며 버그 수정
```
